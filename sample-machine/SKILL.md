# 样机管理 SOP

> 版本：1.0 | 最近更新：2026-04-14
> 用途：样机领用、试用、到期、退款的标准处理流程

---

## 1. 样机管理流程概览

```
飞书审批「样机领用」
        ↓
  产线扫码工序
        ↓
  COROS工厂锁机 webhook
        ↓
  n8n 处理
        ↓
  MES(SN查询)
        ↓
  飞书多维表台账 + 自建PG（主单/任务/设备明细）
        ↓
  到期提醒 → 退还/续期
        ↓
  退款流程（如适用）
```

---

## 2. 样机领用

### 2.1 领用类型

| 类型 | 说明 | 审批流程 |
|------|------|---------|
| 项目问题样机 | PM领用项目问题样机 | 飞书审批 |
| 员工试用机 | 员工内部试用 | 飞书审批 |
| KOL试用机 | KOL外部试用 | 飞书审批 |

### 2.2 领用流程

```
1. PM提交飞书审批「样机领用」
2. 审批通过
3. n8n触发样机处理流程
4. 产线扫码工序（绑定SN）
5. COROS工厂锁机（webhook）
6. 写入飞书多维表台账
7. 写入自建PG（主单/任务/设备明细）
8. 通知领用人
```

### 2.3 相关Workflow

| Workflow | 节点 | 说明 |
|----------|------|------|
| 样机试用 | 58 | 样机试用主流程 |
| 样机试用测试 | 58 | 样机试用测试环境 |
| 样机试用到期提醒 | 15 | 到期提醒 |
| 飞书试用机退款流程 | 92 | 退款流程 |

## 3. 样机试用

### 3.1 试用管理

- **试用期限**：根据审批确定
- **锁机状态**：样机在COROS后台标记为锁机状态
- **SN追溯**：通过SN可查询样机领用记录

### 3.2 到期提醒

**触发时机**：试用到期前N天

**相关Workflow**：
- `样机试用到期提醒` (15节点)

**提醒方式**：
- 飞书消息通知领用人
- 飞书消息通知管理员

**处理选项**：
- 退还：走退还流程
- 续期：重新审批
- 转购买：走购买流程

---

## 4. 样机退还

### 4.1 退还流程

```
1. 领用人发起退还
2. 确认样机状态（SN/锁机/使用情况）
3. 解锁样机（COROS后台）
4. 更新台账状态
5. 更新PG记录
6. 通知管理员
```

### 4.2 退还检查清单

- [ ] 样机外观是否完好？
- [ ] 功能是否正常？
- [ ] 数据是否已清除？
- [ ] 配件是否齐全？
- [ ] 台账状态已更新？

---

## 5. 退款流程

### 5.1 退款场景

- 试用机转购买后退款
- 样机损坏赔偿
- 其他退款场景

### 5.2 退款流程

**相关Workflow**：
- `飞书试用机退款流程` (92节点)
- `Refunds_Warehouse` (54节点)

**处理步骤**：
```
1. 收到退款申请
2. 确认退款原因
3. 查询样机记录（SN/领用人/时间）
4. 确认退款金额
5. 执行退款
6. 更新台账
7. 通知相关人员
```

### 5.3 退款失败处理

**相关Workflow**：
- `退款失败手动退款` (3节点, 非活跃)

**处理步骤**：
1. 检查自动退款失败原因
2. 手动执行退款
3. 记录退款日志
4. 通知申请人

## 6. 样机台账

### 6.1 台账字段

| 字段 | 说明 |
|------|------|
| 主单ID | 唯一标识 |
| SN | 样机序列号 |
| 领用人 | 谁领用的 |
| 领用类型 | 项目/员工/KOL |
| 领用时间 | 什么时候领用的 |
| 到期时间 | 什么时候到期 |
| 状态 | 试用中/已退还/已购买/已退款 |
| 锁机状态 | 已锁/已解锁 |
| 备注 | 其他信息 |

### 6.2 台账查询

```sql
-- 查询样机记录
SELECT * FROM sample_machine WHERE sn = 'xxx' LIMIT 1;

-- 查询到期样机
SELECT * FROM sample_machine 
WHERE status = '试用中' AND expire_date <= NOW() + INTERVAL '7 days';

-- 查询逾期未退还
SELECT * FROM sample_machine 
WHERE status = '试用中' AND expire_date < NOW();
```

## 7. 常见问题

### 7.1 样机领用失败

**可能原因**：
- 审批未通过
- SN不存在
- 锁机失败
- 台账写入失败

**处理步骤**：
1. 检查审批状态
2. 检查SN是否存在
3. 检查COROS后台锁机状态
4. 检查n8n workflow状态
5. 检查数据库写入

### 7.2 到期提醒未发送

**可能原因**：
- 定时workflow未执行
- 飞书Token过期
- 联系人信息缺失

**处理步骤**：
1. 检查`样机试用到期提醒`workflow状态
2. 检查飞书Token
3. 检查联系人信息
4. 手动触发提醒

### 7.3 退款失败

**可能原因**：
- 支付接口异常
- 退款金额错误
- 订单状态异常

**处理步骤**：
1. 检查退款workflow状态
2. 检查支付接口
3. 确认退款金额
4. 手动退款（如需要）
